
# icorp26

<!-- badges: start -->
<!-- badges: end -->

The goal of icorp26 is to support Lehman students
and others in using the CDC RSS data this summer.


## Installation

You can install the development version of icorp26 from [GitHub](https://github.com/) with:

``` r
# install.packages("remotes")
remotes::install_github("elinw/icorp26")
```

## Example

For students, right now the package contains the 
data sets rss1 through rss7.  You should 
be able to access any of this simply by loading
the library

``` r
library(icorp26)
```
